## 代码 

- 方法一：[CocoaPods](https://cocoapods.org/) 导入：`pod 'LPNavPageVC'`

- 方法二：导入`LPNavPageVC`文件夹到你的项目中 (文件夹在 Demo 中可以找到)
* 需要使用的vc请继承于LPPageVC (如果你不懂继承 .. Google)并且实现其相关方法，详情请阅读.h文件;
